This project is a front-end replica of Google's homepage, built as a UI/CSS exercise. It replicates the minimalist design including the Google logo, search input with buttons, top navigation bar, and footer links. Built with pure HTML and CSS (no frameworks), it emphasizes understanding of flexbox/grid layout, box model, and responsive design principles.
Tech used: HTML5, CSS3

Key concepts: Flexbox, responsive design, UI cloning
